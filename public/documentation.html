myFlix

The myFlix application will be showcasing a collection of movies and provide access to information about movies, directors and genres. Users will be able to create an account and manage their personal information. They can also create a list of their favorite movies.

Endpoints

Business Logic URL HTTP Method Request Response
Return list of all movies /movies GET None JSON object holding data about all movies
Return data about movie by title /movies/[title] GET None JSON object holding data a movie
Return data about genre /movies/[genre] GET None JSON object holding data about a genre
Return data about director by name /movies/[director] GET None JSON object holding data director
Registration of new users /users POST JSON object holding data about the user added e.g.
{ username: John, email: john@email.com }		 JSON object holding data about the user added including id e.g.
{ id: 4, username: John, email: john@email.com }
Users can update their username /users/[username] PUT None Text message indicating that the username was successfully
Users can add a movie to their list of favorites /users/favList/[title] PUT None Text message showing that movie was added to list
Users can remove a movie from their list of favorites /users/favList/[title] PUT None Text message showing that movie was removed
Existing users can deregister /users/[id] DELETE None Text message showing that user email was successfully removed
