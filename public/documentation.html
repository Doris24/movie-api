myFlix

The myFlix application will be showcasing a collection of movies and provide access to information about movies, directors and genres. Users will be able to create an account and manage their personal information. They can also create a list of their favorite movies.

Endpoints

Business Logic URL HTTP Method Request body data format Response body data format
Return list of all movies /movies GET None JSON object holding data about all movies
Return data about movie by title /movies/[title] GET None JSON object holding data a movie
Return data about genre /genres/[genre] GET None JSON object holding data about the requested genre
Return data about director by name /directors/[director] GET None JSON object holding data about the requested director
Registration of new users /register POST JSON object holding data about the user added e.g.
{ username: John, email: john@email.com }		 JSON object holding data about the user added including id e.g.
{ id: 4, username: John, email: john@email.com }
Users can update their username /users/[id]/[username] PUT None Text message indicating that the username was successfully
Users can add a movie to their list of favorites /users/[id]/favList/[title] POST JSON object holding data about the movie added e.g.
{ title: Moviename, genre: comedy, director: Johnny, description: Description of the movie }		 JSON object holding data about the movie added including id e.g.
{ id: 222, title: Moviename, genre: comedy, director: Johnny, description: Description of the movie }
Users can remove a movie from their list of favorites /users/[id]/favList/[title] DELETE None Text message showing that movie was removed
Existing users can unregister /users/[id]/unregister DELETE None Text message showing that user email was successfully removed
