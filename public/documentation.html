myFlix

The myFlix application will be showcasing a collection of movies and provide access to information about movies, directors and genres. Users will be able to create an account and manage their personal information. They can also create a list of their favorite movies.

Endpoints

Business Logic URL HTTP Method Request body data format Response body data format
Return list of all movies /movies GET None JSON object holding data about all movies
Return data about movie by title /movies/[title] GET None JSON object holding data the requested movie
Return data about all genres /genres GET None JSON object holding data about all requested genre
Return data about a genre by name /genres/[genre] GET None JSON object holding data about the requested genre
Return data about all directors /directors GET None JSON object holding data about all directors
Return data about a director by name /directors/[director] GET None JSON object holding data about the requested director
Return data about all users /users GET None JSON object holding data about all users
Return data about a user by username /users/[username] GET None JSON object holding data about the requested user
Registration of new users /users POST JSON object holding data about the user added e.g.
{ Username: 'String',
Password: 'String',
Email: 'String',
Birthday: Date }		 JSON object holding data about the user added including id e.g.
{ ID: 4,
Username: John,
Password: 123,
Email: john@email.com,
Birthday: 1989-12-31T23:00:00.000Z, FavoriteMovies: [] }
Users can update their data /users/[title] PUT JSON object holding the changed data about the user e.g.
{ username: 'String',
email: 'String' }		 JSON object holding the data of the user including the updated data
Users can add a movie to their list of favorites /users/[username]/movies/[movieId] POST JSON object holding data about the movie added e.g.
{ title: 'String',
genre: 'String',
director: 'String',
description: 'String' }		 JSON object holding data about the movie added including id e.g.
{ id: 222,
title: Moviename,
genre: comedy,
director: Johnny,
description: Description of the movie }
Users can remove a movie from their list of favorites /users/[username]/movies/[movieId] DELETE None Text message showing that movie was removed.
Existing users can unregister /users/username DELETE None Text message showing that the user was successfully removed.
